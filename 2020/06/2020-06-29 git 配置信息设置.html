1.git 使用前配置信息

在git使用前，需要告诉git你是谁，在向git仓库中提交时需要用到：

1.1配置提交人姓名： git config --global user.name 提交人姓名

ex: git config --global user.name fangjian


1.2配置提交人邮箱: git config --global user.email 提交人邮箱

ex:git config --global user.email fjasmin@sina.com


1.3查看git配置信息：git config --list

ex:

fjasmin@fjasmin-ThinkPad-T460:~$ git config --list

user.name=fangjian

user.email=fjasmin@sina.com

core.autocrlf=input


2.提交步骤

2.1 git init 初始化git 仓库


2.2 git status 查看文件状态


2.3 git add 文件列表 追踪文件


2.4 git commit -m 提交信息 向仓库中提交代码


2.5 git log 查看提交记录


3.撤销

3.1用于暂存区中的文件覆盖工作目录中的文件： git checkout 文件
